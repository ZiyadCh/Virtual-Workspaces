WorkSphere - Virtual Workspace
Description du projet

WorkSphere est une application web interactive permettant de gérer visuellement le personnel sur un plan d’étage. Les utilisateurs peuvent ajouter, déplacer et supprimer des employés selon des règles métier précises liées aux rôles et aux zones autorisées. L’objectif est de fournir une interface moderne, intuitive et responsive pour faciliter l’organisation interne d’une entreprise.

Fonctionnalités principales

Ajout d’un employé via une modale (nom, rôle, email, téléphone, photo, expériences professionnelles).

Répartition du personnel sur 6 zones :

Salle de conférence

Réception

Salle des serveurs

Salle de sécurité

Salle du personnel

Salle d’archives

Restrictions logiques selon le rôle (ex. réception → réceptionnistes uniquement).

Icone pour afficher les donnes d'un employe.

Interface responsive (desktop, tablette, mobile).

Les zones vides obligatoires sont en rouge.

Limitations du nombre d’employés par zone.

Hebergement sur Vercel.
